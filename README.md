# Experiment 4 - Front-End Testing using Jest and React Testing Library

## 📌 Objective

To perform front-end testing in a React application using Jest and React Testing Library, including unit testing, form testing, and snapshot testing.

---

## 🛠️ Technologies Used

* React.js
* Jest
* React Testing Library
* JavaScript (ES6)

---

## 📂 Project Structure

```
src/
  App.js
  Button.js
  Form.js
  Dashboard.js
  App.test.js
  setupTests.js
  App.css
  index.js
```

---

## 🚀 Features Implemented

### 1. Unit Testing

* Tested a Button component
* Verified click event handling

### 2. Form Testing

* Tested input field interaction
* Validated empty input
* Verified success message on submit

### 3. Snapshot Testing

* Tested Dashboard component UI consistency
* Ensured UI structure remains unchanged

---

## ▶️ How to Run the Project

### Install dependencies

```
npm install
```

### Run the application

```
npm start
```

### Run tests

```
npm test
```
